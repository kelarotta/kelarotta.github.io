About

The project played a significant role in the On-the-Job Learning phase of my studies. My part in the project was to create a viewing platform in the VR world of the University of Oulu campus and enable the user to teleport to it. This included designing the platform, coding the TeleportManager script, and creating a simple placeholder user interface.
